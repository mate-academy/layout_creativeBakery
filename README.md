A modern, responsive bakery website designed to present bakery products, company information, contact options, and a blog section.
The project layout adapts to big-desktop, standard desktop, tablet, and mobile screen sizes.

A bakery website featuring four main sections: **Products**, **About Us**, **Blog**, **Contacts Us**.

The header includes the bakery name, site navigation, a short description and a "LEARN MORE" button. There is also a cookie image in the background for decoration.

The navigation menu contains links to each section of the website and a call icon.

The **Products** section shows what the bakery makes, each product block contains: a photo pf the product, the product name, a short description and a button and sign to order the product.

The **About Us** section explains what the bakery is about and highlights key numbers to shom experience. It includes: the foundation year, the number of stores, the number of companies, the number of clients. There is also a cookie crumbs for decoration.

The **Blog** section is located in the footer with a few small icon-links to the bakery`s social media platforms. This keeps footer simple and useful.

The **Contacts Us** section includes: three phone numbers, and an email address.
